## 🚀 Démarrage rapide

```bash
# Installer les dépendances
pip install -r requirements.txt

# Lancer le dashboard BI
python weather_visualizer.py

# Dashboard Streamlit (mode interactif)
streamlit run weather_visualizer.py

# Tests automatisés
test_exercice8.bat
```
